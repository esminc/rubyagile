Welcome, and thanks for visiting Blank CSS.

Every time I design a website, I start with the same basic layout and style my CSS around it. To save time whenever starting a design, I have created some blank website templates with their layouts already completed. I have decided to make these available for free download. You can use these website templates anywhere you want, for anything you want. All I ask is that you link back to Blank CSS in some way.

Each website template that you can download has been slightly styled so you can see the concept of each one. Obviously there is no need to keep this styling and it can be changed to suit your individual needs. No images are used in the styling - it is all done by CSS. This will help decrease your page loading times when using the templates.
